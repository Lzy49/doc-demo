说明: 盒子宽度为800，每个元素设 basis 300 ， shrink 分别为 1，2，3

实际元素宽度应该为：

  • 需要收缩的宽度： 300 * 3 - 800 = 100 ；
  • 经过权重计算后的权重总合：1 * 300 + 2 * 300 + 3 * 300 = 1800
  • 第一个div的缩放宽度为 100 * (1 * 300 / 1800) = 16.67
  • 第二个div的缩放宽度为 100 * (2 * 300 / 1800) = 33.33
  • 第三个div的缩放宽度为 100 * (3 * 300 / 1800) = 50
  • 第一个div的宽度为 300 - 16.67 = 283.33
  • 第二个div的宽度为 300 - 33.33 = 266.67
  • 第三个div的宽度为 300 - 50 = 250
